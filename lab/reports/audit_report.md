## Confirmed Findings from Active Attack Modules

### 1. Broken Object Level Authorization (BOLA)

**Source:** Active attack module

**Severity:** High

**OWASP Category:** API1:2023 Broken Object Level Authorization

**Affected endpoint:** `/vulnerable/inventory/{user_id}`

**Status:** `Confirmed`

**Evidence:**

```json
{
  "attacker_user_id": "731bc566-e0d6-41c8-9819-28f0d73bb3a4",
  "victim_user_id": "28cde77b-b881-48e4-8e4e-76011f618fc8",
  "vulnerable_status_code": 200,
  "secure_status_code": 403,
  "returned_items": [
    {
      "id": "fd80dccf-884b-42b1-ae9b-b04bd5dc42bc",
      "item_name": "Starter Sword",
      "quantity": 1,
      "owner_id": "28cde77b-b881-48e4-8e4e-76011f618fc8"
    },
    {
      "id": "53dae246-5e3d-4ff3-bb13-441dcbf89c1a",
      "item_name": "Golden_AK47_928e2606-327d-44e7-8e05-571d51f44c9b",
      "quantity": 1,
      "owner_id": "28cde77b-b881-48e4-8e4e-76011f618fc8"
    }
  ]
}
```

**Recommended mitigation:**

Validate that the requested user_id matches the authenticated user_id extracted from the token before querying inventory data.

### 2. Broken Object Property Level Authorization / Mass Assignment

**Source:** Active attack module

**Severity:** High

**OWASP Category:** API3:2023 Broken Object Property Level Authorization

**Affected endpoint:** `/vulnerable/users/me`

**Status:** `Confirmed`

**Evidence:**

```json
{
  "initial_profile": {
    "id": "731bc566-e0d6-41c8-9819-28f0d73bb3a4",
    "steam_id": "76561198000000002",
    "username": "Attacker_Test",
    "credits": 100,
    "is_admin": false
  },
  "vulnerable_payload": {
    "username": "MassAssignmentUser",
    "credits": 777777
  },
  "vulnerable_response": {
    "id": "731bc566-e0d6-41c8-9819-28f0d73bb3a4",
    "steam_id": "76561198000000002",
    "username": "MassAssignmentUser",
    "credits": 777777,
    "is_admin": false
  },
  "secure_payload": {
    "username": "SafeMassAssignmentUser",
    "credits": 1
  },
  "secure_response": {
    "id": "731bc566-e0d6-41c8-9819-28f0d73bb3a4",
    "steam_id": "76561198000000002",
    "username": "SafeMassAssignmentUser",
    "credits": 777777,
    "is_admin": false
  },
  "expected_secure_behavior": "username is updated, but credits remains unchanged"
}
```

**Recommended mitigation:**

Use strict input DTOs and whitelist only the fields that the client is allowed to modify. Sensitive fields such as credits and is_admin must never be accepted from client-controlled JSON.

### 3. Weak Token Integrity / User Impersonation

**Source:** Active attack module

**Severity:** Critical

**OWASP Category:** API2:2023 Broken Authentication

**Affected endpoint:** `/users/me`

**Status:** `Confirmed`

**Evidence:**

```json
{
  "attacker_user_id": "731bc566-e0d6-41c8-9819-28f0d73bb3a4",
  "victim_user_id": "28cde77b-b881-48e4-8e4e-76011f618fc8",
  "victim_steam_id": "76561198000000003",
  "status_code": 200,
  "response_body": {
    "id": "28cde77b-b881-48e4-8e4e-76011f618fc8",
    "steam_id": "76561198000000003",
    "username": "Victim_Test",
    "credits": 100,
    "is_admin": false
  }
}
```

**Recommended mitigation:**

Replace unsigned Base64 tokens with signed JWTs or server-side sessions. In a real Steamworks integration, validate the Steam ticket server-side against Steam Web API and generate a signed session token with expiration.

### 4. Client-Side Trust in Transaction Finalization

**Source:** Active attack module

**Severity:** Critical

**OWASP Category:** API6:2023 Unrestricted Access to Sensitive Business Flows

**Affected endpoint:** `/vulnerable/transactions/finalize`

**Status:** `Confirmed`

**Evidence:**

```json
{
  "attacker_user_id": "731bc566-e0d6-41c8-9819-28f0d73bb3a4",
  "initial_credits": 777777,
  "final_credits": 782777,
  "amount": 5000,
  "order_id": "ORDER_FRAUD_bd719afa-b98a-4ed8-b7ea-d798778a0789",
  "init_response": {
    "id": "beebb06c-fcad-45ea-9c2c-751d7b8b89ce",
    "order_id": "ORDER_FRAUD_bd719afa-b98a-4ed8-b7ea-d798778a0789",
    "item_name": "Mega Coins Pack",
    "amount": 5000,
    "approved_by_client": false,
    "finalized": false,
    "owner_id": "731bc566-e0d6-41c8-9819-28f0d73bb3a4"
  },
  "finalize_response": {
    "id": "beebb06c-fcad-45ea-9c2c-751d7b8b89ce",
    "order_id": "ORDER_FRAUD_bd719afa-b98a-4ed8-b7ea-d798778a0789",
    "item_name": "Mega Coins Pack",
    "amount": 5000,
    "approved_by_client": false,
    "finalized": true,
    "owner_id": "731bc566-e0d6-41c8-9819-28f0d73bb3a4"
  },
  "secure_status_code": 403
}
```

**Recommended mitigation:**

Never trust approved_by_client or any payment state sent by the client. The backend must verify the transaction directly with Steam Web API or a trusted payment provider before granting credits or virtual items.
