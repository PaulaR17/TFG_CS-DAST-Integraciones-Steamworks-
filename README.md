# TFG - DAST Auditoría Steamworks Integrations

## Objetivo
Desarrollar una plataforma DAST para evaluar vulnerabilidades en integraciones Steamworks dentro de entornos controlados.

## Arquitectura
Cliente (Unity) → Backend simulado → Proxy → Módulos DAST → Generador de informes

v0.0 - Estructura inicial creada
